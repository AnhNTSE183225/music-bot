# One regex per line. Lines starting with # are ignored.
# Blocks titles like: 99999 Doa Hong, 9 Doa Hoa Hong, 9 Đóa Hoa Hồng (and mixed diacritics).
\b9+\s*[dđ](?:[oóòỏõọôồốổỗộơờớởỡợ])(?:[aáàảãạăằắẳẵặâầấẩẫậ])\s*(?:h(?:[oóòỏõọôồốổỗộơờớởỡợ])(?:[aáàảãạăằắẳẵặâầấẩẫậ])\s+)?h(?:[oóòỏõọôồốổỗộơờớởỡợ])ng\b
# Blocks titles containing "Rap Viet" / "Rap Việt" in any casing.
\brap\s*vi(?:e|ệ)t\b
# Blocks titles containing the word "cover" in any casing.
\bcover\b
# Blocks titles containing "Siêu Hài" (or "Sieu Hai") in any casing.
\bsi(?:e|ê)u\s*h(?:a|à|á|ả|ã|ạ)i\b
# Blocks titles containing "kawaii kutegomen" in any casing.
\bkawaii\s*kutegomen\b
# Blocks titles containing the word "kejo" in any casing.
\bkejo\b
